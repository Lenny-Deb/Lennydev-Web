-About Me-

Hi, I'm Lenny_Dev, but my real name is Ian. I'm an indie game dev from Spain who started to make games because I loved the idea of creating my own worlds. I usually use Godot 4 for my projects, but I'm also very familiarized with RPG Maker and Game Maker Studio.

I work ALL by myself, I don't have any publishers of programmers hired, I'm a single person who likes to create videogames in their room, I handle all my webs, social media accounts, everything. So yeah, I'm kind of real.

I only do 2D games (for now), but I would like to try some 3D stuff, I kind of used Cinema 4D it to make FNaF animations a while ago. In my games, I also do ALL of the work, programming, drawing, composing, everything. I like to have the control of my stuff.
